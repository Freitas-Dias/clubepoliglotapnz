## 📖 Sobre o projeto

O **Clube Poliglota PNZ** foi fundado em **agosto de 2024** pelo Prof. Ricardo de Freitas Dias, docente da Universidade de Pernambuco – Campus Petrolina. O projeto nasceu do desejo de democratizar a prática de idiomas para estudantes universitários e membros da comunidade local.

Este repositório contém o código-fonte do site oficial do clube — uma página HTML estática, 100% gratuita, hospedada via **GitHub Pages**.

## 🚀 Como publicar (deploy)

### Opção 1 — GitHub Pages (recomendado)

```bash
# 1. Clone o repositório
git clone https://github.com/Freitas-Dias/clubepoliglotapnz.git

# 2. Acesse a pasta
cd clubepoliglotapnz

# 3. Adicione o arquivo HTML renomeado como index.html
# (já está configurado)
```

Depois:
1. Acesse **Settings → Pages**
2. Em *Source*, selecione **main branch**
3. Clique em **Save**
4. Seu site estará em: `https://freitas-dias.github.io/clubepoliglotapnz`

### Opção 2 — Netlify Drop

1. Acesse [netlify.com/drop](https://netlify.com/drop)
2. Arraste o arquivo `index.html`
3. Receba o link imediatamente
